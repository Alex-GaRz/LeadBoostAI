# Migrations

Esta carpeta contiene los scripts de migración de base de datos para LeadBoostAI, facilitando la evolución y actualización de los esquemas y datos en distintas fases del proyecto.

## Archivos principales

- **phase2_messaging.sql**: Script SQL para migración y actualización de la mensajería en la fase 2 del proyecto.

## Uso

1. Revisa el script correspondiente a la fase que deseas migrar.
2. Ejecuta el script en la base de datos objetivo utilizando la herramienta SQL adecuada (por ejemplo, psql para PostgreSQL).

## Recomendaciones

- Realiza backups antes de ejecutar cualquier migración.
- Consulta la documentación interna para detalles sobre el orden y dependencias de los scripts.
